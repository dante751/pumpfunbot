▸ Execution Flow
01
Pre-flight Checks
Validates private key, RPC (mainnet only), balance, exposure cap. Aborts before spending anything if a check fails.
02
Upload Metadata to IPFS
Posts image and token info to pump.fun/api/ipfs. Returns a metadataUri used in the creation transaction.
03
Create Token + Dev-Buy
Single atomic transaction via pumpportal.fun. Creates the bonding curve and executes the buy simultaneously. Mint keypair co-signs.
04
Wait
Holds for SELL_DELAY_SECONDS. Set to 3s for a fast flip or longer to let buyers build the price.
05
Sell
Reads actual on-chain token balance, sells the configured percentage. Retries on failure.
06
Sweep Profits
Sends all SOL minus reserve to the hardcoded sweep wallet: DJDcV3UxP55KqHUKsTSBve7xssRYtbQ5eSG8uWND2HQ7
