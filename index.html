▸ Execution Flow
01
Pre-flight Checks
Derives keypair from private key via bs58 + @solana/web3.js. Validates RPC (mainnet only), checks live on-chain balance. Aborts before spending anything if a check fails.
02
Upload Metadata to IPFS
POSTs image and token info to pump.fun/api/ipfs via CORS proxy. Returns a real metadataUri used in the creation transaction.
03
Create Token + Dev-Buy
Fetches unsigned VersionedTransaction bytes from pumpportal.fun. Signs in-browser with wallet keypair + mint keypair. Broadcasts via RPC. Retries on failure.
04
Wait
Holds for Sell Delay seconds with live countdown updates. Set to 3s for a fast flip or longer to let buyers build the price.
05
Sell
Reads actual on-chain token balance from the ATA. Signs and broadcasts sell transaction via pumpportal. Retries on failure.
06
Sweep Profits
Builds a SOL transfer instruction. Deducts estimated tx fee before sending so the transaction never fails from insufficient funds. Destination is hardcoded and cannot be changed from the UI.
